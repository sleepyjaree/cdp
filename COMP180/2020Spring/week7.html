Review of Week 7: Intro to machine learning¶

Reading material¶

  • The SciKit Learn tools are a collection of programs and utilities designed for predictive data analysis. The material on the website is a bit advanced and we’ll use the judiciously and in context with the course’s objective. Nonetheless, the site contains some good material, including a very useful glossary of terms, and a thorough but dense introduction to Naive Baye classifiers which we’ll use during week 7.
  • Technical documentation for the Gaussian Naive Bayes classifier.
  • The notebook with the introductory implementation of a Naive Bayes Classifier.
